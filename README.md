
# 🎓 EduCast - Content Broadcasting System

A modern, role-based school Content broadcasting system that allows teachers to schedule content and principals to review and approve live displays. 

---

## 🚀 Getting Started

### 1. Installation
```bash
# Clone the repository
git clone https://github.com/Manthanpelne/content-broadcasting-system-frontend

# Install dependencies
npm install

# Run the development server
npm run dev
